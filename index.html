1. 目的

{{APP_NAME}}（以下「本アプリ」）は、Meta Threads API を利用し、投稿の自動化・返信管理・インサイト取得を支援します。本ポリシーは、本アプリにおける個人情報および関連データの取扱いを定めるものです。

2. 収集・利用するデータ

  • コンテンツ情報：投稿本文、返信本文、コンテンツID、親子関係ID、タイムスタンプ。
  • インサイト/メタデータ：投稿・ユーザーの指標（例：インプレッション、いいね数、リプライ数、リポスト数、引用数 など）。
  • 技術情報：OAuthトークン（アクセストークン/更新トークン）、実行ログ、エラーログ、IPアドレス等（セキュリティ・運用目的）。
  • 運用付帯情報：スケジュール、下書き内容、紐づくスプレッドシート上のデータ。

3. 利用目的

  • Threads への投稿・返信の作成/管理（自動化含む）
  • インサイトの取得・分析・レポーティング
  • 品質改善、障害対応、セキュリティ対策、法令・規約遵守

4. 保存期間

データは運用目的の達成に必要な期間のみ保持し、不要となった場合は速やかに削除します。標準では最長12か月を上限とし、法令によりより長い保存が求められる場合を除きます。

5. 第三者提供

本人の同意または法令に基づく場合を除き、第三者に提供しません。ただし、運用上の委託先（例：Google Apps Script / Google スプレッドシート / ホスティング等）に対して、機密保持契約等の適切な管理の下で取り扱いを委託することがあります。

6. 安全管理措置

  • アクセストークンは安全なストレージ（例：GAS プロパティ等）で暗号化もしくはアクセス制御の下に保存
  • 最小権限・レート制御・アクセスログの監査
  • 通信のHTTPS化、公開鍵ピンニング/検討、依存ライブラリの脆弱性対応

7. ユーザーの権利と手続

本アプリが保持するデータの開示・訂正・利用停止・削除を希望される場合、データ削除手順ページをご参照の上ご連絡ください。合理的な範囲で速やかに対応します。

8. 13歳未満の方の利用

本アプリは13歳未満の方を対象としていません。該当するデータの取得が判明した場合、速やかに削除します。

9. ポリシーの変更

内容を改定する場合、本ページの改定日を更新して告知します。重要な変更は合理的な方法で周知します。
